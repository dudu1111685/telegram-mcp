## ✨ תכונות

*   **🔄 לולאה אינסופית**: ה-AI לא רק שולח הודעה ונעלם. הוא נכנס ל"מצב טלגרם" שבו הוא מחכה לפקודות שלך, מבצע אותן, ומדווח חזרה - ללא הפסקה.
*   **🔘 כפתורים חכמים**: כל הודעה מגיעה עם כפתורים מותאמים להקשר (למשל: "הרץ טסטים", "תקן באג") לתגובה מהירה בלחיצה אחת.
*   **🗣️ תמיכה רב-לשונית**: מדבר בשפה שלך! אם תכתוב בעברית, הוא יענה בעברית.
*   **🎨 עיצוב עשיר**: שולח הודעות Markdown מעוצבות עם טקסט מודגש, בלוקים של קוד, ורשימות.
*   **🔌 תמיכה אוניברסלית**: עובד עם **Claude Desktop**, **Claude Code**, **Cursor**, ו-**Google Antigravity (Gemini IDE)**.

## 🚀 התחלה מהירה

### 1. דרישות מוקדמות: הגדרת טלגרם 📱
לפני ההתקנה, עליך להכין את סביבת הטלגרם:

1.  **צור בוט**:
    *   פתח את [@BotFather](https://t.me/BotFather) בטלגרם.
    *   שלח `/newbot` ועקוב אחר ההוראות כדי לקבל את ה-**Bot Token** שלך.
2.  **צור קבוצה (Supergroup)**:
    *   צור קבוצה חדשה.
    *   **הפעל נושאים (Topics)**: לך להגדרות הקבוצה > היסטוריית צ'אט > הפעל "Topics" (או "Forum"). *זה קריטי!*
3.  **הוסף את הבוט כמנהל**:
    *   הוסף את הבוט החדש לקבוצה.
    *   תן לו הרשאות **Admin** מלאות (במיוחד "Manage Topics").
4.  **אתחול**:
    *   שלח הודעת סתמי בנושא "General" של הקבוצה (למשל: "Init").
    *   *זה מבטיח שה-ID של הקבוצה יהיה נגיש.*
5.  **קבל את מזהה הקבוצה (Group ID)**:
    *   העבר הודעה מהקבוצה ל-[@RawDataBot](https://t.me/RawDataBot) (או השתמש בשיטה אחרת) כדי למצוא את ה-**Chat ID** (בדרך כלל מתחיל ב-`-100`).

### 2. התקנה
אנו מספקים מתקין אוטומטי שמגדיר הכל עבורך.

```bash
python3 install.py
```

הסקריפט יבצע:
1.  התקנת תלויות Python.
2.  בקשת **Telegram Bot Token** ו-**Group ID**.
3.  הגדרה אוטומטית של **Claude Desktop**, **Claude Code**, ו-**Antigravity**.

### 3. הפעלת "מצב טלגרם"
כדי להתחיל את לולאת העבודה מרחוק, פשוט אמור ל-AI שלך:

> **"אני יוצא לקפה, תמשיך מטלגרם."**

ה-AI יבצע:
1.  יפתח נושא (Topic) חדש בקבוצת הטלגרם שלך.
2.  ישלח לך הודעה: *"אני מוכן. מה לעשות?"*
3.  **ימתין** לתשובה שלך (חוסם את הריצה עד שתענה).

### 4. סיום הסשן
כדי לצאת מהלולאה, פשוט לחץ על הכפתור **"תודה, נמשיך מהמחשב"** בטלגרם.

## 📂 קבצי קונפיגורציה

*   **`CLAUDE.md`**: חוקים קשוחים ל-Claude Code לאכיפת הלולאה והעיצוב.
*   **`.cursorrules`**: חוקים לזיהוי אוטומטי ב-Cursor וסוכנים אחרים.
*   **`TELEGRAM_MODE.md`**: פרומפט מערכת שניתן להדביק ידנית במידת הצורך.

## 🛠️ הגדרה ידנית (אופציונלי)
אם אתה צריך להגדיר IDE אחר ידנית, השתמש בקונפיגורציית MCP זו:

```json
{
  "mcpServers": {
    "telegram-agent": {
      "command": "/usr/bin/python3",
      "args": ["/absolute/path/to/telegram_mcp/server.py"]
    }
  }
}
```
